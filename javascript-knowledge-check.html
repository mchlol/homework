What are the eight data types in JavaScript?

number e.g. let x = 123; This can be an integer or floating point (decimal) number.
string any text, including single characters e.g let x = "hello";
boolean e.g. let x = true; or let x = false;
bigInt - for numbers too big for the number type.
e.g let x = 1233567890123356789012335678901233567890n - where n creates a bigInt.
undefined - where a variable has been created, but not assigned a value e.g. let x;
null - where the value of the variable is not known or equal to nothing.
object - the only data type that is not primitive. It contains properties, made up of key/value pairs.

            
                let myObject = {
                    name: "mchlol",
                    age: 32,
                    skillLevel: beginner
                    }
Functions also belong to the object data type.
symbol a special property included in objects. Read more info about symbols here.


Which data type is NOT primitive?

Object, it can contain multiple types of data in properties, where other data types can only contain one type of data.

What is the relationship between null and undefined?

Null and undefined both technically mean "nothing" but only undefined can refer to an existing variable that exists but hasn't been assigned a value.

What is the difference between single, double, and backtick quotes for strings?

A string can use single quotes ' or double quotes " but cannot be used at the same time. Escape a quote by using the backslash \. Backticks ` can create strings with embedded variables.


What is the term for embedding variables/expressions in a string?

Concatenate, or join.


Which type of quote lets you embed variables/expressions in a string?

Backtick `


How do you embed variables/expressions in a string?

By using backticks.

    let myName = "mchlol"
    let x = `Hello, my name is ${myName}, nice to meet you.`
    
    x = "Hello, my name is mchlol, nice to  meet you."


How do you escape characters in a string?

By using a backslash \ before the character.

        let x = "I\'m making an example";
        
        x = "I'm making an example"

            let x = 123;
                let y = x.toString();
                console.log(y);
            
            x = "123";


What is the difference between slice/substring/substr?

.slice() can be used to extract part of a string, it takes two parameters - the first is the index to start at, and the second is the index to end at. The start is included in the extraction, but the end is not. Using a negative will count from the END of the string.
.substring() similar to .slice() but cannot take negatives.
.substr() can be used for the same thing as .string() but while the first parameter is still the index to start at, the second parameter is how many indexes after the starting index to end at.

What are the three logical operators and what do they stand for?

&& - AND: Returns true only if the result of all cases is true
|| - OR: Returns true as long as at least one case results in true
! - NOT: Returns true if result is false.

What are the comparison operators?

== - equality, able to perform type conversion
=== - strict equality, must be the same type
< and> - less than or greater than
<= and>= - less than or equal to, and greater than or equal to

What are truthy and falsy values?

A truthy value is any value that can be equal to 1, or true, a falsy value is equal to 0, or false.

What are the falsy values in JavaScript?

0


What are conditionals?

Conditionals are performed to make decisions based on the boolean outcome of a conditional test. E.g. if true run this code, if false run this code.

What is the syntax for an if/else conditional?

        
            if (condition === true) {
                return true;
                } else {
                return false;
                }
            }

What is the syntax for a switch statement?

        
            let x = true;
            let result;
            switch(x) {
                case true:
                result = "yes";
                break;
                case false:
                result = "no";
                default:
                result = "not sure"
            }

What is the syntax for a ternary operator?

        
            (condition) ? code to run if true : code to run if false;

What is nesting?

If you want to test another condition after one has been met, you can nest another condition within it. e.g.

            
                if (condition === true) {
                    if (anotherCondition === true) {
                        return true;
                    }
                } else {
                    return false;
                }
