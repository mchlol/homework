Fundamentals Part 1

Name the three ways to declare a variable?

let, const, and var

Which of the three variable declarations should you avoid and why?

var is an outdated way to declare a variable, as it is globally scoped. This can cause issues especially when using multiple JavaScript files.

What rules should you follow when naming variables?

The hard rules are to not start with a number, and the only special characters allowed are $ and _. You cannot use reserved words such as class, let, return, or function.
It is also best practice to use camelCase, give variables obvious, descriptive (human-readable) names (avoid things like let abc = "Joe" ) and keep them consistent throughout your code.

What should you look out for when using the + operator with numbers and strings?

The + operator performs both addition and concatenation (joining). When using two numbers, + will add them together. When using two strings, + will concatenate them. When using a number and a string, + will concatenate them unless the string is first converted to a number.

How does the % operator work?

The % or 'modulo' operator is used to calculate the number left over from a division. E.g. 2 % 2 will equal 0 as there is nothing left over.

Explain the difference between == and ===.

=== is the strict comparison operator, meaning the operands being compared must be of the same type. Using == also compares, but performs type coercion, so should be mostly avoided as it can produce bugs.
E.g. 2 == "2" will return true, but 2 === "2" will return false because the values are not the same type - 2 is a number and "2" is a string.

When would you receive a NaN result?

If you were to try to perform a mathematical equation using variables that are not numbers, e.g. "Two" * 5 will result in NaN because "Two" is a string, not a number.

How do you increment and decrement a number?

Using the increment counter ++ or decrement counter --. You can also increment a variable immediately with +=. 

            let apples = 2;
                    apples++ // 2 + 1 = 3
                    
                    apples += 2 // 3 + 2 = 5

Explain the difference between prefixing and post-fixing increment/decrement operators.

Postfixing counter++ and prefixing ++counter both do the same thing. If we intend immediately use the increased or decreased value, we would use the prefix form ++counter. If we would like to increase the value but start off using the original value, we use the postfix form counter++.

What is operator precedence and how is it handled in JS?

Operator precedence determines the order in which operators in an expression will execute. There is a precedence table that outlines the priority value of all of the operators, the one with the highest number is executed first. E.g. 1 + 2 * 2 will first perform addition because the + operator has higher precedence than *.

How do you access developer tools and the console?

Depending on your browser you can use a keyboard shortcut or through the menu: for Chrome this would be cmd + option + i or menu > more tools > developer tools.

How do you log information to the console?

Using the helpfully descriptive console.log("Print this!")

What does unary plus operator do to string representations of integers?

If you have values as strings, "one" and "two", they will be treated as strings, not numbers, so the unary plus operator (+) will concatenate (join) them together as a string, like so: "onetwo"
If the values are "1" and "2" these are still strings, not numbers, so you will still get "12" (a string), even if one is actually of type number (like "1" and 2). You can only end up with 3 if both values are of type number, ie. 1 and 2.

Fundamentals Part 2

What are the eight data types in JavaScript?

number e.g. let x = 123; This can be an integer or floating point (decimal) number.
string any text, including single characters e.g let x = "hello";
boolean e.g. let x = true; or let x = false;
bigInt - for numbers too big for the number type.
e.g let x = 1233567890123356789012335678901233567890n - where n creates a bigInt.
undefined - where a variable has been created, but not assigned a value e.g. let x;
null - where the value of the variable is not known or equal to nothing.
object - the only data type that is not primitive. It contains properties, made up of key/value pairs.

            
                let myObject = {
                    name: "mchlol",
                    age: 32,
                    skillLevel: beginner
                    }
Functions and arrays also belong to the object data type.
symbol a special property included in objects. Read more info about symbols here.


Which data type is NOT primitive?

Objects are not primitive - they can contain multiple types of data in properties, where other data types can only contain one type of data.
Functions and arrays are objects.

What is the relationship between null and undefined?

Null and undefined both technically mean "nothing" but only undefined can refer to an existing variable that exists but hasn't been assigned a value.

What is the difference between single, double, and backtick quotes for strings?

A string can use single quotes ' or double quotes " but cannot be used at the same time. Escape a quote by using the backslash \. Backticks ` can create strings with embedded variables.


What is the term for embedding variables/expressions in a string?

Template literal


Which type of quote lets you embed variables/expressions in a string?

Backtick `


How do you embed variables/expressions in a string?

By using backticks.

    let myName = "Bob"
    let x = `Hello, my name is ${myName}, nice to meet you.`
    
    x = "Hello, my name is Bob, nice to  meet you."


How do you escape characters in a string?

By using a backslash \ before the character.

        let x = "I\'m making an example";
        
        x = "I'm making an example"


What are methods?

Methods are built in functions in Javascript that can be used to perfom common tasks, such as convert a number to string, convert a string to upper or lowercase, or return the type of a variable. 

            let x = 123;
                let y = x.toString();
                console.log(y);
            
            x = "123";


What is the difference between slice/substring/substr?

.slice() can be used to extract part of a string, it takes two parameters - the first is the index to start at, and the second is the index to end at. The start is included in the extraction, but the end is not. Using a negative integer for the first argument will count from the END of the string.
.substring() similar to .slice() but cannot take negatives.
.substr() can be used for the same thing as .string() but while the first parameter is still the index to start at, the second parameter is how many indexes after the starting index to end at.

What are the three logical operators and what do they stand for?

&& - AND: Returns true only if the result of all cases is true
|| - OR: Returns true as long as at least one case results in true
! - NOT: Returns true if result is false.

What are the comparison operators?

== - equality, able to perform type conversion
=== - strict equality, must be the same type
< and > - less than or greater than
<= and >= - less than or equal to, and greater than or equal to

What are truthy and falsy values?

A truthy value is any value that can be equal to 1, or true, a falsy value is equal to 0, or false.
Is -1 also falsy?

What are the falsy values in JavaScript?

False, 0, -0, an empty string, null, undefined, and NaN.
NaN is not equivalent to anything, including another NaN.


What are conditionals?

Conditionals are performed to make decisions based on the boolean outcome of a conditional test. E.g. if true run this code, if false run this code.
It is usually possible to use only one value to determine true of false, e.g. if (a) will return true if the variable a has a truthy value.

What is the syntax for an if/else conditional?

        
            if (condition === true) {
                return true;
                } else {
                return false;
                }
            }

What is the syntax for a switch statement?

        
            let x = true;
            let result;
            switch(x) {
                case true:
                result = "yes";
                break;
                case false:
                result = "no";
                default:
                result = "not sure"
            }

What is the syntax for a ternary operator?

        
            (condition) ? code to run if true : code to run if false;

What is nesting?

If you want to test another condition after one has been met, you can nest another condition within it. e.g.

            
                if (condition === true) {
                    if (anotherCondition === true) {
                        return true;
                    }
                } else {
                    return false;
                }

Fundamentals Part 3

What are functions useful for?

Functions are actions performed by your code that can be used over and over again.

How do you invoke a function?

Invoke a function by calling it with functionName(). The parentheses perform the action of actually calling the function - the function will not run without them.

What are anonymous functions?

Anonymous functions are functions that are used as part of an expression, their return value is used immediately and the function itself is not used again.

What is function scope?

Function scope refers to the variables with a functions code block only being accessible within the function itself. If a variable is defined within a function, it cannot be accessed anywhere else.

What are return values?

Return values are the products of a function when it is called. For example if you want to run a function that adds two numbers together, the return value will be the result of the addition.

What are arrow functions?

An arrow function is a shorter more concise way of writing a function expression.
