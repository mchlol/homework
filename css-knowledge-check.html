CSS Basics

What are the main differences between external, internal, and inline CSS?

External CSS, where the css file is linked to in the head, can be used to style multiple pages linked to one CSS file, so it can serve as the style rules for multiple pages.
Internal CSS, where the css is contained within a style tag in the head, can be used to style the elements on that page only.
Inline CSS, where styles are defined within an element's tag, can only be used to style individual elements one at a time.

What is the syntax for class and ID selectors?

Class selectors are accessed using .classname and ID selectors are accessed using #idname.

How would you apply a single rule to two different selectors?

You can apply the same rule to multiple elements by putting a comma between them 

                .firstclass, .secondclass {
                    color: #FF0000;
                    }
This will apply the color to both the .firstclass and .secondclass.

Given an element that has an id of title and a class of primary, how would you use both attributes for a single rule?

You can chain the selectors together to apply one rule: 

                #title.primary {
                    color: #000000;
                    }
This will only select the element that has both the #title id and the .primary class.

What does the descendant combinator do?

The descendant combinator can select an element that is the child or descendant of a specific parent or ancestor. 

.parent .child {
                color: #FF000;
                }
The ancestor goes first, then a space, then the descendant.

  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)

Between a rule that uses one class selector and a rule that uses three type selectors, which rule has the higher specificity?

The class selector rule would be applied first as it takes precedence over any number of type selectors.

Block and Inline Elements

What is the difference between a block element and an inline element?

A block element will break normal flow and always sit on its own line, where an inline element will remain in the normal flow of the page.

What is the difference between an inline element and an inline-block element?

An inline element can only have its width adjusted, where an inline-block element can adjust both width and height.

What does the box-sizing css property do?

The box-sixing property can affect whether the width and height of an element refer to its contents only, or also include the padding.

Is an h1 block or inline?

H1, h2, and all other header elements are block elements.

Is span block or inline?

Span is an inline element.

Is button block or inline?

Buttons are block elements.

Is div block or inline?

Divs are block elements.

Would you use margin or padding to create more space between 2 elements?

Margins affect the space between elements.

Would you use margin or padding to create more space between the contents of an element and its border?

Padding affects the space between an elements contents and its border.

CSS Units

Why would you want to use em or rem for font-size instead of px?

I would not use em for font size, because each nested element using em could end up getting bigger and bigger. I would use rem as the font size would remain consistent to the body font size, and would be more responsive than using px.

What are some instances where you might want to use vh and vw?

These units are good for setting the size of elements relative to the size of the viewport, such as a hero section or even sidebar.

What are some instances where you might want to use px instead of a relative unit?

It could be beneficial to use px for padding and margin for smaller elements such as buttons or small card-like components, but I would use em for larger elements to keep the space consistent with the element font size.

Text Styles

What are the 2 ways to add fonts that are not installed on a user's computer?

The easiest way is to use an external API such as Google Fonts, where the font can be linked in the head of the HTML or imported into the first line of CSS.
Another way - which often loads faster - is to download a font and import it into your CSS via url using @font-face. 

@font-face {
            font-family: cool-font;
            src: url(../fonts/cool-font-file.woff);
                  }

What is the "system font stack" and why would you want to use it?

The system font stack lists several common fonts, and the browser will display the first one the user has installed on their computer. This is a good way to avoid using the brower's default font, which may not match your design.

Which property would you use to increase or decrease the space between letters in a word?

To increase the space between letters in a word, use the letter-spacing property.

Which property would you use to increase or decrease the space between lines in a paragraph?

To increase the space between lines of text vertically, use the line-height property.

More CSS Properties

Which property would you use to make an element transparent?

The opacity property is used to make an element transparent, for example opacity: 0.5; would make an element 50% opaque.

Which property would you use to make a background image tile?

To make a background image tile, add repeat to the background shorthand property, or set it specifically using background-repeat: repeat. Setting background-repeat: no-repeat will prevent the background image from tiling.

Which property would you use to add scrollbars to an element?

If the content of an element is overflowing it's container, use overflow: scroll to give the container a scrollbar. This will apply on both the x and y axis.

Which property would you use to add a shadow behind an element?

box-shadow is used to add a shadow behind an element; it typically is used with four values, offset-x, offset-y, blur-radius, and a colour.e.g. 2px 2px 4px black would add 2px to the right, 2px to the bottom, and 4px of blur in black. Using negative values would, for example, shift the offset-x to the left instead of the right.

Which property would you use to create rounded corners on an element?

To round the corners of an element, use border-radius. Using this with a single value will round all corners to the same size, or all the corners can be set to different radius values.

How would you use border-radius to make a circular element?

To create a circular element, the element must be a square with the same height and width, then use border-radius: 50% to round all the corners into a circle.

More Selectors

What is the difference between the child combinator and the descendant combinator?

The child combinator .parent > .child selects the direct child of an element, the descendant combinator .parent .child will select any child or grandchild of an element.

How does the syntax of pseudo-classes and pseudo-elements differ?

Pseudo classes are targeted with one colon :, pseudo-elements are targeted with two colons ::.

Do pseudo-classes exist somewhere in HTML? Do pseudo-elements?

Pseudo-classes do exist in HTML. Pseudo-elements do not, and are only generated by the CSS for display purposes.

Name two ways you could select every second child of an element, starting with the first.

  1. element:nth-child(2)
  2. element:nth-child(2n+1)

What is the difference between div:first-child and div :first-child? What will each select?

div:first-child will select the first div.
div :first-child will select the first element that is the child of a div.

What selector would you use to style a button a user is currently hovering over? How about one that is currently being clicked on?

Style a button that is being hovered over with button:hover and a button that is being clicked with button:active.

How could you select all input elements with a type of text?

input[type="text"] will select every input with the attribute type with a value of text.

How could you select all classes that begin with thunder ?

class^="thunder"], where the caret (^) after the attribute targets any element beginning with the characters that follow. Similarly $ targets text that ends in those characters, and * will target the characters anywhere in the attribute.

Positioning

What is the difference between static and relative positioning?

Static is the default position property, these elements just follow the normal flow of the html document.
Relative positioning also follows the normal flow, but an element with position: relative can also be moved relative to it's own position in normal flow, without affecting the other elements - they behave as if the element was still where it's supposed to be. Positioning is done the the top, left, bottom, and right properties, set to any value in absolute or relative units eg 10px, 1em, etc.

What is absolute positioning useful for?

An element with position: absolute can be positioned relative to the next parent element set to position: relative or absolute. If there isn't one, the element will be positioned relative to the html element. These elements are removed from normal flow and other elements will behave as if it was never there.

What is the difference between fixed and sticky positioning?

A fixed element is positioned relative to the viewport, and will stay in view even when the page is scrolled. A sticky element will also stay in place, but only while it's parent is in view. Once the parent has scrolled out of view, the sticky element will scroll away with it.

CSS Functions

What are the four CSS math functions we covered above?

  • calc(): for performing basic addition, subtraction, multiplication, and division. Can be used to dynamically size elements or fonts based on the size of other elements, or custom properties (variables).
  • min(): used for sizing an element or font, and can take several values in different units, e.g. min(50%, 400px). As the browser is resized, min() will use whichever value is smallest.
  • max(): like min(), but will take whichever value is largest. min() and max() can both also perform basic math.
  • clamp(): takes three values - the minimum, ideal, and maximum. The element will never be smaller than it's lowest value, or larger than it's highest value.

How do we use CSS math functions in our CSS?

CSS math functions are primarily used for sizing, which can provide an alternative to using media queries.

How can CSS functions help make websites and applications more responsive?

CSS functions are useful for responsive design because sizing properties such as height, font-size, padding, etc. can be calculated based on the size of other elements or of the viewport, without having to hard-code all the properties.

Custom Properties

How would you declare a custom property with a name of text-color?

First create a variable name, starting with two hyphens, and set it to any valid CSS value.


            .class {
                 --text-color: #000;
            }

How would you access a custom property with a name of background-color?

Access the custom property by calling the variable as a value to a CSS property.


            .class {
                background-color: var(--background-color);
            }

Where would you declare a custom property to have its scope be global and accessible by all other selectors?

To create a global custom property, create it in the :root element (which is similar to the html element but with higher precedence). To scope the custom property to a specific class or element, create it within that class or element. It cannot be used anywhere else.

Where would you declare a custom property so that a user's theme setting from their OS or browser was taken into account?

A media query, such as prefers-color-scheme, which can detect if the user has dark mode set and the custom property will be applied accordingly.

Frameworks & Preprocessors

What are some advantages of using a CSS framework?

CSS frameworks such as Tailwind and Bootstrap make it quick and easy to code UI for websites and applications, even without design skills. They can also make it easy for a team to collaborate because everyone is using the same code.

What are some disadvantages of using a CSS framework?

CSS frameworks can be difficult to debug, especially if you're using one because your own CSS skills are not that strong. They can also be complicated to customise as the framework needs to be overrided. Finally, frameworks can cause a lot of websites to look the same, if developers are all using the same frameworks.

What are some advantages of using a CSS preprocessor?

CSS preprocessors such as Sass and Stylus can make it easy to write more powerful CSS using features such as mixins, variables, and loops. One helpful feature is nesting your CSS meaning you don't have to worry as much about knowing all the different kinds of selectors.
Preprocessors also make your processed CSS code DRY-er, and more organised - meaning it is easier to read and understand by other developers.

What are some disadvantages of using a CSS preprocessor?

The output CSS can be difficult to debug as it doesn't exactly match what was input.
Having to learn and set up additional tools can take up too much extra time.
CSS now includes solutions to some of the problems that preprocessors were invented to solve, such as variables and mixins.
Once a project has been built with a preprocessor, it can be difficult to remove it.
