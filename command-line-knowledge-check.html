Command Line Basics

What is the command line?

The command line is where you can provide instructions directly to your computer, such as searching for and creating files or opening programs.

How do you open the command line on your computer?

I’m using macOS, so I can go to Applications > Utilities and open Terminal from there (as a shortcut, I have it pinned it to my dock).

How can you navigate to a particular directory?

cd <directory path> will navigate to that specific directory.

Where will cd on its own navigate you to?

Calling cd on its own will take you back to the home directory.

Where will cd .. navigate you to?

Calling cd .. will take you to the parent directory of the directory you’re in.

How do you display the name of the directory you are currently in?

Find out which directory you’re in by calling pwd which stands for ‘present working directory’.

How do you display the contents of the directory you are currently in?

Display the contents of the directory you’re in by calling ls to show a basic list of files, ls -a to include hidden files (where A is for All), or ls -l to show the list of files including their details such as size and type (L is for Long form listing). You can also change how this list is displayed by chaining, e.g. ls -lh will display the list with human-readable sizes instead of the default number of bytes (such as 1.7k and 35m).

How do you create a new directory?

mkdir <filename> (make directory)

How do you create a new file?

touch <filename>

How do you destroy a directory or file?

rm <filename> (remove)

How do you rename a directory or file?

mv a.txt b.txt will rename a.txt so that it becomes b.txt.
