Drop-down Menus

Steps to creating a drop down menu

The goal: Click on a nav-bar or a button, and the menu slides down nicely.

  1. Create a nav bar with a list of 4 links.
  2. Create a CSS class with visible/hidden which can be toggled.
  3. Write a function that toggles the class when clicked.
  4. Create a CSS keyframes animation to handle the sliding.
