Intro to HTML & CSS

What do HTML and CSS stand for?

Hyper-Text Markup Language, and Cascading Style Sheets.

Between HTML and CSS, which would you use for putting paragraphs of text on a webpage?

HTML adds content such as paragraphs of text onto a webpage.

Between HTML and CSS, which would you use for changing the font and background color of a button?

CSS is for style such as changing fonts and colours.

Elements and Tags

What is an HTML tag?

Tags tell the browser what kind of content the element will contain, so it knows how to interpret it.

What are the three parts of an HTML element?

An opening tag, content, and a closing tag.

HTML boilerplate

What is the purpose of the doctype declaration?

The doctype declaration tells the browser which version of HTML to use.

What is the HTML element?

The HTML element is the root of the webpage, and contains all of the elements.

What is the purpose of the head element?

The head element contains all of the meta- information about the webpage such as the title, character encoding, and links to external stylesheets. Nothing that needs to be visible is put in the head, only in the body.

What is the purpose of the body element?

The body element contains the actual content of the webpage to be displayed, such as text and images.

Working with Text

How do you create a paragraph in HTML?

<p>Paragraph text content goes here!</p>

How do you create a heading in HTML?

For a top level heading, <h1>Heading</h1>

How many different levels of headings are there and what is the difference between them?

There are six levels of headings, and these are used the same way as in essays or articles to indicate the hierarchy of the content.

What element should you use to make text bold and important?

strong

What element should you use to make text italicized to add emphasis to it?

em (for emphasis)

What relationship does an element have with any nested elements within it?

An element that has nested elements within it is the parent of those elements (which become children).

What relationship do two elements have if they are at the same level of nesting?

Two elements at the same level of nesting are siblings.

How do you create HTML comments?

<!-- comment text would go here -->

Lists

What HTML tag is used to create an unordered list?

ul will create a list of bullet points.

What HTML tag is used to create an ordered list?

ol will create a list with consecutive numbering.

What HTML tag is used to create list items within both unordered and ordered lists?

li will create a list item, and is the same within an unordered or an ordered list. 


    <ul>
        <li>This is a list item</li>
        <li>This is another list item</li>
    </ul>

Links and Images

What element is used to create a link?

<a>

What is an attribute?

An attribute is used to tell the browser more information about the type of content your tag will include.

What attribute tells links where to go to?

href, e.g. <a href=“http://www.google.com”>

What is the difference between an absolute and relative link?

An absolute link includes the entire path of a link, whereas a relative link contains only part of the path. The former is usually used to link to an entirely different website. The latter is used when linking to files within the same directory.

Which element is used to display an image?

<img>

What two attributes do images always need to have?

An image tag must always include a src attribute, to point the browser to the correct file. It also must always include an alt attribute, to provide further info about the image in case the image cannot be loaded or the user is using a screenreader.

How do you access a parent directory in a filepath?

Using two dots will direct the file path to the folder directly above the current one, e.g. ../filename.jpg

What are the four main image formats that you can use for images on the web?

Jpg (or jpeg), png, gif, and svg. There is also webp but it's not so widely used yet.
