FizzBuzz

Write a program that takes a user's input and prints the numbers from one to the number the user entered. However, for multiples of three print Fizz instead of the number and for the multiples of five print Buzz. For numbers which are multiples of both three and five print FizzBuzz.

Understanding the Problem

Take in user input as a positive integer. Set a counter variable starting from 1. Begin counting up to the number the user has entered. For every number along the way that is a multiple of 3, print Fizz instead of the number. For every multiple of 5, print Buzz. For every number that is a multiple of both 3 and 5, print FizzBuzz. This should look something like 1, 2, Fizz, 4, Buzz, 6, 7, 8, 9, 10, 11, 12, 13, 14, FizzBuzz.

Plan

Pseudocode

Create a button for the user to click.
Create a div to hold our display.
Assign a function that opens a prompt when the button is clicked.
Store the prompt input in a variable.
Check that the prompt input is a positive integer.
If it's not, encourage the user to pick something we can use.
Limit the number to 20.
Create a counter variable.
Write a function that includes a for loop that will increment the counter variable, up until it reaches the input.
Write a function to print each number in the loop to the div.
Within this function, nest an if loop to check if the value meets the fizz/buzz/fizzbuzz conditions and if so, print that instead of the value.

Interface

Output:

Click the input button above to choose a number, and begin the counter.
Numbers will display here!
