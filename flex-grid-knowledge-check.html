Knowledge Check - Flexbox & CSS Grid

Homework for The Odin Project

Flexbox

What are the 3 values defined in the shorthand flex property? (e.g. flex: 1 1 auto)?

Flex grow, flex shrink, and flex basis (min, max, and ideal size).

flex: initial is the same as setting flex: 0 1 auto.

What is flex grow?

Flex grow determines if the flex item can grow to fill the available space. If an item has a larger flex grow property than the other items, it will be bigger.

What is flex shrink?

Flex shrink determines if the flex item can shrink when the container shrinks, and how much. If the flex shrink property is larger on one item than the others, it will shrink more.

What is flex basis?

Flex basis sets the initial or ideal size of a flex item. If set to auto, it will look at the set height or width, or the size of the content. If set to 100%, it will take up the entire space of the parent container, either height or width depending on the main axis (row or column).

How do you make flex items arrange themselves vertically instead of horizontally?

You can arrange flex items vertically instead of horizontally by changing the rule on the flex container to flex-direction: column.

In a column flex-container, what does flex-basis refer to?

Flex basis will refer to the height of the flex item.

In a row flex-container, what does flex-basis refer to?

Flex basis will refer to the width of the flex item.

Why do the previous two questions have different answers?

When flex direction is set to column, the main axis is vertical and the cross axis is horizontal. When flex-direction set to row, the main axis is horizontal and the cross axis is vertical.

What is the difference between justify-content and align-items?

Justify-content will organise the items along the main axis, and align-items organises them along the cross axis.

How do you use flexbox to completely center a div inside a flex container?

Use justify-content to center along the main axis and align-items to center along the cross axis.

What's the difference between justify-content: space-between and justify-content: space-around?

Space-between will organise items with even space between them, and space-around will organise items with even space between them AND on either side.

How can you use Flex to make a two-dimensional layout?

Create a two-dimensional layout with Flex by allowing your flex items to wrap.

        
            div {
                display: flex;
                flex-flow: row wrap;
            }

CSS Grid

Why was CSS Grid introduced?

CSS Grid was introduced to allow finer grained control over layouts similar to magazine and newspaper layouts.

Which CSS layout module would you use to easily make equal sized items in a container?

It is easier to use CSS Grid because the size of the items is declared at the time of creating the grid, where flex children size is declared on the child itself.
It's also easier to use fr units to make all child items the same size.

Creating a Grid

How does an HTML element become a grid item?

An HTML element will become a grid item when it's parent is set to display: grid.

What is the space between lines on the grid?

The spaces between lines on the grid are the grid tracks.

How do you set gutters (also known as alleys) in the grid?

Gutters/alleys can be set by using the gap property.

Describe what happens when you have more content than defined tracks.

When there are more grid items than defined in the parent grid container, they create an implicit grid.

How could you change the size for those undefined tracks?

Define the size of implicit grid items by giving the parent grid container a grid-auto-rows or grid-auto-columns property.

Positioning Grid Elements

Explain the difference between a track and a line.

A track is the actual content in a row or a column. A line is the divider between the rows and between columns.

What is the smallest unit on a grid?

The smallest unit on a grid is a cell. E.g. the content betwen row line 1 and 2 and column line 1 and 2 is the first cell.

What kind of value do we give to the grid-column-start or grid-column-end properties?

These properties take single integer values e.g. grid-column-start: 2 indicates the content will be positioned starting from column line 2.

Which property can we use to combine all the start and end values for a grid item?

We can combine all the start and end values to position a grid item with grid-area, which will take 4 integers divided by forward slashes to position the item, in this order: row start, column start, row end, column end.

    
        .item {
            grid-area: 1 / 1 / 3 / 4;
        }

Which grid container property can map out a visual structure of grid items?

The grid-template-areas property, set on a grid container, can create a visual structure of grid items, e.g.

                
        .container {
            grid-template-areas:
                header  header
                sidebar article
                footer  footer;
        }

Advanced Grid Properties

How do you create several grid tracks of the same size without manually typing each one out?

Using the repeat() function, eg. grid-template-columns: repeat(3, 200px) will create 3 columns that are all 200px wide.

What is the difference between a static and dynamic size value?

A static value, such as px will remain that size at all times. A dynamic value, such as a percentage, vw or fr value, will change according to the size of the window/viewport.

How can you assign a grid track a flexible value that changes depending on the remaining space available in the grid?

The fr is a flexible value that can change based on the space available.

How can you assign grid tracks an uneven distribution of the remaining space in a grid?

By using the fr unit, eg. grid-template-columns: 1fr 2fr will create 2 columns, with the second being twice the width of the first, regardless of the size of the container itself.

Which CSS functions will return the smallest or largest value supplied to them?

The min() function will return the smallest value passed to it, and the max() function will return the largest value passed to it.

Which CSS Grid-only function allows you to supply a minimum and maximum track size that is calculated in realtime?

The minmax() function, when passed to the repeat() function, can determine the smallest and largest value possible for an element which changes based on the size of the window or viewport.

Which global CSS function allows you to supply a minimum, ideal, and maximum value that is calculated in realtime?

The clamp() function takes three values in this order; the minimum size, ideal size, and maximum size. The appropriate size is calculated based on the size of the container or viewport as it is resized.

What attribute of repeat() can be used to fill in as many grid tracks as possible, given certain constraints?

auto-fill can be used to continue to add items/cells to the grid as it is resized, even if they are empty.

What is the difference between auto-fit and auto-fill?

auto-fit will fit the existing items into the available space, and will stretch them to fill the whole space. auto-fit does not change the size of existing items, but will add more items as soon as there is room for them.

Flexbox vs Grid

When might you use Flexbox over Grid?

Flexbox is useful for one dimensional layouts when you need to organise items in one row or column.

When might you use Grid over Flexbox?

Grid is more useful than Flexbox for complete layouts and more fine control over columns and rows at the same time.

When might you use the two of these tools together?

Grid items can be flex parents, and flex items can be grid parents.
