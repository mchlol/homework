Linting

What is linting?

Linting is using an automated tool to scan your code for syntax errors and style rules as you go. It's a little bit like a spelling and grammar checker, but for your code.

Which problems can linting prevent?

Linting can catch errors before they become bugs. E.g. you could set up your linter to change all variable declarations from var to let, or even to change all variable declarations to const to avoid the risk of overwriting a variable or polluting the global namespace. It can pick out missing semi-colons or braces, or notify you when you have declared a variable but not used it.

Why should you use Prettier?

Prettier differs from a linter in that it does not check for errors, but automatically fixes things like spacing and indentation. It does what it says on the tin - makes your code prettier. This makes it much more human readable.
